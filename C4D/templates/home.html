Overview

This application currently loads everything in to one large flattened table of Raw Land Records. In the future this data can be normalized in to Land Records, Properties, Offices, Municipalities, etc for greater searchability and indexing.

  • Search for a group of records or individual records on Search page
  • Edit land records after finding necessary record via Search page
  • Import files and review recents imports on Import Page
  • Review all land records and import logs on Admin page
